# 为什么选择 UniCore？

### 综合BIM与GIS能力

根据我们针对行业相关解决方案的调查分析显示，目前这方面的竞品如广\*达，采用的是只基于 BIM 图形引擎模仿 GIS 功能，只是简单加载一部分的底图，无法达到专业 GIS 图形引擎的要求。老\*云则采用行业普遍采用的解决方案，即 BIM 系统使用 BIM 图形引擎，GIS 系统使用 GIS 图形引擎，不仅导致开发者需要学习和维护两套系统，还额外增加开销，使用者的体验也无法保证，BIM 功能与 GIS 功能是割裂的，无法自由切换使用。

UniCore （下称通用图形引擎）独特地将 BIM 与 GIS 功能融合于一身。通用图形引擎`打破了原先 BIM 与 GIS 这两种系统相互独立的局面`，提供了一个统一的解决方案，使得处理复杂的建筑和地理数据变得前所未有的简单。

### 高效的性能

相较于我们原先研发的 BIM 图形引擎，我们通过自研 3DTiles 格式转换算法，结合 GIS 图形引擎加载 3DTiles 模型的性能优势，实现了我们之前无法做到的性能优化。

原先的 BIM 图形引擎使用 glTF 模型进行展示，在性能羸弱的集显电脑是卡顿甚至无法运行的。市面上的竞品如广*达，同样存在无法运行的问题。通用图形引擎经过测试，能够在性能羸弱的集显电脑打开原本无法打开的模型，且显示效果更好。
